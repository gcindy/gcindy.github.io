项目目录结构

文件或目录 说明
.dart_tool 记录了一些dart工具库所在的位置和信息（运行环境&配置）
.idea android studio 是基于idea开发的，.idea 记录了项目的一些文件的变更记录
android Android项目文件夹
build 构建过程的文件和缓存，一般不需要手动编辑这个目录的内容**
ios iOS项目文件夹
lib 存放我们的dart语言代码（开发页面放lib文件夹下）
linux linux项目文件夹
macos macos项目文件夹
test 用于存放我们的测试代码
web web端项目文件夹
windows windows项目文件夹
.gitgnore git忽略配置文件
.metadata IDE 用来记录某个 Flutter 项目属性的的隐藏文件
analysis_options.yaml 分析dart语法的文件，老项目升级新项目有警告信息，则可以删除此文件
pubspec.lock 当前项目依赖所生成的文件
pubspec.yaml 当前项目的一些配置文件，包括依赖的第三方库、图片资源文件等
xxx.iml 工程文件的本地路径配置 (xxx代表项目名称)
README.md READEME文件
