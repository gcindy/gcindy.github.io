List

  • length 长度

  • isNotEmpty 是否不为空

方法

  • add 增加数据
  • addAll 拼接数据（增加多个数据）
  • indexOf 查找
  • remove(传入具体的值) 删除数据
  • removeAt(传入索引的值) 删除数据

  • insert(index,value) 指定位置插入数据
  • insertAll(index,list) 指定位置插入 List
  • toList 其他类型转换为 List
  • join List 转为字符串
  • split 字符串转换为 List
