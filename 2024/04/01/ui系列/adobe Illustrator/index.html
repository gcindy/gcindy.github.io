Last updated on April 22, 2024 pm

图层移动

  • 上移一层：command + [

  • 下移一层：command + ]

  • 置顶图层：shift + command + [

  • 置底图层：shift + command + ]

快捷键

  • shift 单边等比缩放
  • shift + option 中心等比缩放

这里的快捷键是我自己自定义的

  • 圆形：o

  • 矩形：R

  • 多边形：D

  • 星形：S

  • 直线：L

  • 钢笔：P

  • 选择工具：V

  • 直接选择工具：A

  • 复制：command + option

  • 魔棒工具：Y

  • 形状生成器：shift + M

  • 剪刀：C
