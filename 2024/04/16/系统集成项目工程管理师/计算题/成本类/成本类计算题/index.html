成本类计算题

本文最后更新于 2024年4月22日 上午

挣值分析

  • 计划价值(Planned Value, 简称 PV)：指截止某个时间点计划要完成工作量的价值， 即计划要做的事情的价值。

  • 挣值(Earned Value, 简称 EV)：指截止某个时间点实际已经完成的工作量的价值，即实际做了事所产生的价值。

  • 实际成本(Actual Cost, 简称 AC)：指截止某个时间点实际已经发生的成本，即花了多少钱。

  • 进度偏差(Schedule Variance, 简称 SC)：

    • SV > 0，进度超前
    • SV < 0，进度落后

  • 成本偏差(Cost Variance, 简称 CV)：

    • CV > 0，成本节约
    • CV < 0，成本超支

  • 进度绩效指数(Shedule Performance Index, 简称 SPI)：

    • SPI > 1，进度超前
    • SPI < 1，进度落后

  • 成本绩效指数(Cost Performance Index, 简称 CPI)：

    • CPI > 1，成本节约
    • CPI < 1，成本超支
偏差指标 采取措施
进度落后，成本超支
- 用高效人员代替低效人员
- 在预防风险的情况下并行施工
- 提高资源利用
- 加强、改进沟通，提高效率
- 尽可能一次性把事情做对，减少返工
- 加强沟通
- 增强优质资源
- 外包和缩小项目范围
进度落后，成本节约
- 赶工(例如全体加班方式)加快进度
- 使用高效资源来替换低效资源加快进度
- 改进方法，提高工作效率
进度超前，成本节约
- 抽调部分人员用于其他项目
- 加强质量控制，密切监控项目
- 必要时调整计划或基准等方法改进，或者改变相关计划
进度超前，成本超支
- 整个项目需要抽出部分人员以放慢工作进度
- 整个项目存在成本超支现象濡要采取控制成本措施
- 项目中区分不同的任务,采取不同的成本及进度措施
- 必要时调整成本
- 优化施工方案、提高效率、加强质量管理减少返工、加强沟通，以降低成 本
- 在确保进度按期完成的基础上,可以降低进度以节约成本
- 总结项目进度“提前"的经验，并记录下来，把这经验传播到项目的其他班 组,甚至其他项目或未来的项目
